慢病

基于 UKPDS 框架的 5 状态 Markov 模型，整合 BMI、HbA1c、低血糖事件的微观模拟。预算影响分析覆盖 1000 万患者人群。

常见问题

关于平台的常见问题解答

平台支持哪些 HEOR 模型？

支持 Markov 模型（队列/微观）、决策树、微模拟、分区生存模型 (PSM)、冰山模型等 18 种建模方法。每种方法都有对应的引擎模块，可独立运行或组合使用。

校准是如何工作的？

平台支持贝叶斯校准和最小二乘校准。上传观察数据后，系统自动拟合模型参数，并输出 RMSE、拟合优度和校准覆盖图。校准结果会与模型参数一起版本化保存。

输出可以直接用于 NICE/ICER 申报吗？

可以。平台输出的审阅包包含 NICE 和 ICER 要求的标准图表（CEAC、PSA 散点图、龙卷风图、队列轨迹）和元数据。Enterprise 用户可以自定义输出模板以匹配特定 HTA 机构要求。

支持哪些数据格式？

支持 CSV、Excel (.xlsx)、JSON、SAS (.sas7bdat) 和 R (.rds) 格式。平台会自动识别字段类型并进行数据校验，确保时间粒度和事件定义符合建模要求。

有 API 访问吗？

有。平台提供完整的 RESTful API（OpenAPI 3.0 规范），支持程序化提交建模任务、查询结果和导出报告。API 文档可在 api-docs.html 查看。

数据安全如何保障？

所有数据传输使用 TLS 1.3 加密，存储使用 AES-256 加密。平台支持数据驻留选项（美国、欧盟、亚太），符合 HIPAA 和 GDPR 要求。每次运行的数据在 30 天后自动清除，用户可随时手动删除。

Starter、Professional、Enterprise 有什么区别？

Starter（免费）提供示例流程和基础 Markov 模型。Professional（$99/月）解锁完整建模流水线和所有引擎。Enterprise（$499/月）增加多国 HTA 模板、自定义导出、专属客户经理和 SLA 保障。详见 pricing.html。

支持多国 HTA 吗？

支持。平台内置 12 个国家/地区的 HTA 模板、成本数据和支付方数据。Professional 用户可进行单国分析，Enterprise 用户可进行多国对比分析并生成各国特定格式的申报材料。
