# HEOR Modeling Platform

卫生经济学与结局研究建模平台

## 核心能力

- Markov模型构建
- 成本效果分析
- 预算影响分析
- 敏感性分析
- 概率敏感性分析

## 快速开始

    git clone https://github.com/MoKangMedical/heor-modeling-platform.git
    cd heor-modeling-platform
    pip install -r requirements.txt
    python src/main.py --model markov
