REST API 文档

HEOR Modeling Platform 后端API完整参考。基础URL: http://localhost:8000/api/v1

证据管理

POST /projects/{project_id}/evidence

上传证据数据，平台会自动校验并生成证据对象。

参数类型说明
namestring证据名称
typeenumkm_survival | survival_table | hazard_table
pointsarray数据点数组 [{seq_no, time_value, estimate_value}]
POST /api/v1/projects/default/evidence Content-Type: application/json { "name": "NSCLC PFS - KEYNOTE-010", "type": "km_survival", "points": [ {"seq_no": 1, "time_value": 0, "estimate_value": 1.0}, {"seq_no": 2, "time_value": 1, "estimate_value": 0.968} ] }
GET /projects/{project_id}/evidence

获取项目下的所有证据对象列表。

POST /evidence/{evidence_id}/validate

验证证据数据的完整性和一致性。

概率函数

POST /probability-functions

从证据对象生成概率函数。

参数类型说明
evidence_iduuid证据对象ID
cycle_lengthnumber周期长度 (默认1)
cycle_unitenummonth | year | week
distributionenumweibull | exponential | log_normal

运行管理

POST /runs

创建模拟运行。

参数类型说明
model_typeenummarkov | partitioned_survival | microsimulation
cyclesinteger周期数 (默认60)
initial_populationinteger初始人群 (默认1000)
sample_sizeintegerPSA样本量 (默认1000)
discount_ratenumber折现率 (默认0.03)
GET /runs/{run_id}/results

获取运行结果，包括队列轨迹和成本效果分析。

分析

GET /analytics/icer/{run_id}

计算增量成本效果比 (ICER)。

POST /analytics/psa/{run_id}

执行概率敏感性分析 (PSA)。

参数类型说明
n_simulationsinteger模拟次数 (默认1000)
sampling_methodenumlhs | monte_carlo
GET /analytics/ceac/{run_id}

获取成本效果可接受曲线 (CEAC) 数据。
