这也是结果页的默认阅读顺序: 先回答“值不值得”，再回答“稳不稳”，最后回答“能不能交付”。

默认阅读顺序

一次 run 结束后，先回答“值不值得”“拟合够不够好”“能不能直接用于汇报”。

结果页不会只给你一堆图。它会先把 ICER、QALY、阈值判断和校准状态放在最前面，然后再进入 PSA 散点图、CEAC、队列轨迹和产物列表。

先确认模型是否贴近真实观察数据

默认展示观察值对预测值覆盖图，并给出 RMSE 或拟合优度，帮助你判断校准是否可接受。

不确定性

用 PSA scatter 和 CEAC 解释不确定性

不是只给一个 ICER，而是直接告诉你样本分布和在阈值下的成本效果可接受概率。

队列轨迹

看患者队列什么时候开始明显转移

动态 Markov 与 cohort trace 会一起出现，帮助你判断状态流动是否符合临床逻辑，并直接导出审阅包。
