从你的目标开始 Entry Points

访问者通常不是来“看能力模块”的，而是想立刻完成一件具体工作。

所以首页先告诉你从哪里进入最合适，而不是先让你消化平台内部术语。

Start with Evidence

我手里有一份 KM 或 survival 数据，想先把它整理干净。

适合先验证字段、统一时间粒度，并把零散表格变成可复用对象。

预计 2-3 分钟 输出：证据对象
进入证据工作台
Start with Calibration

我想向团队演示 observed vs predicted 校准是怎么跑出来的。

适合直接看参数边界、异步任务、overlay 和 best-fit 参数如何回流到模型。

预计 3-4 分钟 输出：校准结果
进入校准工作台
Start with Review

我需要一页能讲清楚结果的界面，用来审阅、汇报或演示。

适合直接展示动态 Markov、scatterplot、cohort trace 和可下载产物。

预计 2 分钟 输出：结果页
进入结果审阅页
先整理证据 把 KM、hazard、survival 数据变成可复用对象
再生成可运行函数 让每次 run 都调用同一层概率定义
然后跑模拟 保留样本矩阵、版本信息和运行轨迹
最后解释结果 用 overlay、scatterplot、cohort trace 和 artifacts 讲清楚
